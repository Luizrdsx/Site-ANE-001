## Stack Tecnológico<br>
- Estrutura Frontend: HTML5 (Semântico)
- Estilização: Tailwind CSS (v3.4 via CDN) + CSS Personalizado
- Interatividade: JavaScript Vanilla (ES6+)
- Ícones: FontAwesome 6.4.0
- Fontes: Poppins (Google Fonts)
- Serviços Externos:
  - FormSubmit (Envio de emails para contato/matrícula)
  - ViaCEP (API de autocompletar endereço)
  - Elfsight/SnapWidget (Placeholder para Feed do Instagram)<br><br>
